# angular2-seed
Angular2 seed project to help to run angular2 out of the box


# How to start

- `cd resouces`

- run `npm install`

- run `npm run start`


# Demos

* [Basic](http://localhost:3000/index.html)
* [Form & Form validation](http://localhost:3000/form.html)
* [Router](http://localhost:3000/router.html)
